# Airbnb PriceCheck

A Data Analysis project focused on exploring Airbnb listing prices, cleaning real-world data, and building interactive insights using Python and Power BI.

---

## 📌 Project Overview

This project analyzes Airbnb listing data to uncover pricing trends, location-based insights, room-type distributions, availability patterns, and other important business metrics.

The goal of this project is to:
- Practice real-world data cleaning
- Perform Exploratory Data Analysis (EDA)
- Build meaningful visualizations
- Create an interactive Power BI dashboard
- Showcase Data Analyst skills for portfolio and job applications

## 📂 Project Workflow

### 1️⃣ Data Collection
- Imported Airbnb dataset in CSV format

### 2️⃣ Data Cleaning
- Handled missing values
- Removed duplicates
- Fixed data types
- Cleaned price-related columns
- Managed null review dates

### 3️⃣ Exploratory Data Analysis (EDA)
Performed analysis on:
- Price distribution
- Room types
- Availability
- Review counts
- Neighborhood trends
- Host activity

### 4️⃣ Data Visualization
Created visual insights using:
- Bar charts
- Line charts
- Heatmaps
- KPI cards
- Location analysis

### 5️⃣ Power BI Dashboard
Built an interactive dashboard for:
- Price analysis
- Listing performance
- Neighborhood comparison
- Room type trends

---

## 📊 Key Insights

- Entire homes/apartments generally have higher average prices
- Certain neighborhoods show premium pricing trends
- Listings with more reviews often indicate better occupancy performance
- Availability strongly affects pricing behavior
- Price data contains noticeable outliers
